# 🌙 Lunar Smoke Heart

A beautiful animated web project featuring a glowing rotating moon, twinkling stars, and a dynamic smoke heart created using HTML, CSS, and JavaScript.

## ✨ Features

- 🌕 Realistic rotating moon with lighting effects
- ✨ Twinkling starfield background
- 💨 Animated smoke particles forming a heart shape
- 🎥 Smooth animations using requestAnimationFrame
- 📱 Responsive design

## 🚀 Technologies Used

- HTML5
- CSS3 (animations, gradients, shadows)
- JavaScript (Canvas API)

## 📸 Preview

A calming night-sky scene with a glowing moon at the center and a floating smoke heart animation.
